# Network Scanner

A Python-based network scanner for port scanning and network reconnaissance.

## Features

- TCP port scanning with concurrent threading
- Domain name resolution
- Flexible port specification (ranges, lists)
- Fast scanning with configurable workers

## Usage
```bash
python main.py
```

## Requirements

- Python 3.9+
- Standard library only (no external dependencies)
